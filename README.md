# Sudoku Puzzle Generator
Olivia Hau

This program generates a valid sudoku board, and allows the user to play a game of Sudoku.

To generate the board, run the program. The finished board will print in the output. When asked if you want to play, type Y if you want to play. Enter the number you want to insert into the board, row location, and column location when prompted.

The board is firstly generated by filling in the diagonal boxes (going from top left to bottom right). Then, the rest of the spaces are filled in recursively until a valid board is found.
